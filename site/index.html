Bewerbung einreichen

Häufig gestellte Fragen

  • Was ist das Kernangebot von code-agenten.ch?

    Wir liefern Ihnen AI-Werkzeuge (Coding-Agents), die aktiv an Ihrem Code mitschreiben, Bugs fixen und Reviews erstellen. Dazu gibt es Schweizer Hosting und Hilfe bei der Einrichtung für Ihr Team.

  • Brauche ich zusätzliche AI-Abos (z. B. ChatGPT Plus)?

    Nein. Der Zugriff auf leistungsstarke Coding-Modelle (GLM-4.7, Kimi K2.5, DeepSeek 3.2, MiniMax M2.5, Qwen 3.5, etc.) ist bei uns inklusive. Sie zahlen nur ein Abo für alles.

  • Was unterscheidet code-agenten.ch von Claude Code oder GitHub Copilot?

    Der Datenschutz. Bei US-Tools verlassen Ihre Daten den Schweizer Rechtsraum oder sind durch Gesetze wie den US Cloud Act trotzdem ausländischen Behörden ausgeliefert. Bei uns bleibt jeder Schnipsel Code auf Schweizer Servern. Zudem bieten wir persönlichen Support und Schulungen direkt vor Ort an.

  • Was passiert mit meinem Code? Wird der für AI-Training genutzt?

    Nein. Ihr Code bleibt Ihr geistiges Eigentum. Wir garantieren vertraglich, dass Ihre Daten ausschliesslich für Ihre eigene Instanz genutzt und niemals zum Training anderer Modelle verwendet werden.

  • Welche Sprachen werden unterstützt?

    Alles Gängige: Java, Python, TypeScript, C#, Go und mehr. Da die KI auf OpenCode basiert, versteht sie fast jedes Projekt sofort.

  • Können die Agents in unsere Pipeline (CI/CD)?

    Ja. Die Agents können als automatisierte Instanz in Ihren Review-Prozess eingebunden werden, um Pull-Requests vor der finalen Freigabe auf Logikfehler oder Sicherheitsrisiken zu prüfen.

  • Helfen Sie uns bei der Einführung?

    Wir begleiten Sie aktiv. Von der technischen Anbindung bis zur Schulung Ihrer Entwickler im effizienten Umgang mit Coding-Agents sorgen wir dafür, dass die Produktivitätsgewinne in Ihrem Unternehmen nachhaltig ankommen.
